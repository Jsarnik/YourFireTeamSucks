YourFireTeamSucks.com is a way for you to search recent destiny activities you or the players you searched have participated in. Matches are calculated by querying the Destiny API for the selected platform, event and users, then matching similar activities.

All results are from Bungie post game carnage reports, ranking is determined by a custom algorithm driven off of weighted values chosen by you, the user.
